- Tic Tac Toe (Python)

A console-based Tic Tac Toe game built using Python.

-- Features --
- Two-player game
- Randomized first player
- Win and draw detection
- Board layout guide for players

-- Skills Demonstrated --
- Python functions
- Loops and conditionals
- User input handling
- Program structure

-- How to Run -- 
python tic_tac_toe.py
